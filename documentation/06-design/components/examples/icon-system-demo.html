Implementation Examples

Basic Button with Icon

Status Indicator

<span class="status-indicator" role="status" aria-label="User account is active"> <svg class="icon icon--xs icon--success" aria-hidden="true"> <use href="/icons/svg-sprite.svg#icon-check-circle"></use> </svg> <span aria-hidden="true">Active</span> </span>

Table Header with Sort Icon
